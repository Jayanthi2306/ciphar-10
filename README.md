# CIFAR-10 Image Classification

## Description
Implemented a convolutional neural network from scratch to classify images in the CIFAR-10 dataset.

## Approach
- Custom CNN using PyTorch
- Data augmentation with random horizontal flips
- Trained and evaluated on CIFAR-10 dataset

## Results
- Achieved ~75% test accuracy

## How to Run
1. Install dependencies  
   pip install -r requirements.txt
2. Run training  
   python train.py
